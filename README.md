# llm agent

A small Python package scaffold for building an LLM-focused command-line tool.

## Getting Started

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the package:
   ```bash
   python -m llm_agent.main --prompt "Hello from LLM agent"
   ```

## Testing

```bash
pytest
```
